# PDFs

Place your source book PDF in this folder.

This project was built for **_Hands-On Machine Learning with Scikit-Learn, Keras,
and TensorFlow_** by Aurélien Géron (O'Reilly, 2019), but it works with any text
PDF — just drop a file here and point `PDF_PATH` in your `.env` at it (or leave it
blank and the ingester will pick up the first `*.pdf` it finds in this folder).

> The book PDF itself is **not** included in this repository because it is
> copyrighted material. Supply your own legally obtained copy.
